# Human-in-the-Loop Escalation Pattern

## Problem

Some outcomes require human validation for risk, legal, or customer impact.

## When to Use

Use for low-confidence responses or high-impact transactions.

## When Not to Use

Do not use for deterministic low-risk automations.

## Tradeoffs

Improves safety but can slow resolution times.
